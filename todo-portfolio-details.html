  • Project URL: Todo List App

This project is a deployed ReactJS To Do List App.

A few of the tools I implemented in this project were:

  • - Saving states (React's useState)
  • - Using that state to display desired UI content.
  • - Running functions as and when desired (React's useEffect)
  • - Components
  • - Props
  • - Preventing default behaviors


Users Can:

  • - Add items to their to-do list
  • - Mark items as completed or uncompleted (with corresponding UI elements)
  • - Remove items from their to-do list
  • - Filter which items are shown on the list - filter options: All, Completed, Incomplete
