Glass Profile

  • Project URL: Glass Website

This project was primarily an exercise in CSS, with the intention of creating a one-page, clean professional profile with links to socials and a contact button.

The UI and design are intended to create a glassy effect that is light and bright, and easy on the eyes.

This design can be purposed as a great virtual business card, and would make an attractive landing page to be scanned from a QR code.
