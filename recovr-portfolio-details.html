RecoVR is a Virtual Reality Meditation App that provides a calm, peaceful outdoor environment in which to meditate.

Developers: Valerie Nierenberg & Kyle Whitten

Features:

  • - Music and Ambient Sounds + Guided Meditations
  • - Hand Presence
  • - Particle System mapped to control triggers to synchronize with breathing
  • - Detailed terrain environment with lake
  • Built For: Oculus Quest 2
  • Built With: Unity 2020.3.10, Photon Pun 2, Unity Collaborate, C#, GitHub, Visual Studio Code, Oculus Developer Hub, Oculus Quest 2
