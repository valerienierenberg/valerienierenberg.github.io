Augmented Reality Web App

  • Project URL: AR Web App (Use your phone)

For this project, I created an augmented reality javascript web app using ar.js.

The functionality of the app is marker-based, and if you visit the project repo linked above, you can find two markers to scan in order to see the 3D cubes.

This is a proof-of-concept prototype that can be applied to future projects.

I have big plans for next steps with this, so stay tuned!
